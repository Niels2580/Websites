Send e-mail with PHP

How to use

  • We do not handle GET requests
  • Value of your action attribute in your form should be https://tm-statweb.be/sendmail.php
  • POST variable 'studentemail' is used to send email to, without this POST var set or if the value is not valid the e-mail will not be send (set this input to hidden)
  • POST variable 'subject' is used as e-mail subject, standard is 'No subject'
  • All POST variables used in the form will be listed in the email

Please use and don't abuse sendmail.php, this is for educational purposes only.

Test form

Configuration

Send mail when form is submitted to this address

Optional fields

Example

Always include 'studentemail' in your form else the mail will not be send.

<form method="post" action="https://tm-statweb.be/sendmail.php">
<div>
<label for="gebruikersnaam">Gebruikersnaam:
<input type="email" id="gebruikersnaam" name="gebruikersnaam">
</label>
</div>
<div>
<label for="wachtwoord">Wachtwoord:
<input type="password" id="wachtwoord" name="wachtwoord">
</label>
</div>
<input type="hidden" name="studentemail" value="niels@tm.be">
<div>
<input type="submit" value="Aanmelden">
</div>
</form>

Want to do it yourself?

Copy below code and make a new file sendmail.php on your server. Change the variables and run the page. Do not forget to setup your mailbox first.

<?php
$to = "receiver@mail.com";
$subject = "E-mail subject";
$message = "Hello world!";
$headers = "From: yourname@yourdomainname.be"
mail($to, $subject, $message, $headers);
?>

Get POST variables in PHP

https://www.php.net/manual/en/reserved.variables.post.php or be lazy and type $whatever = $_POST["name"], the attribute name from the HTML will be used.

If needed use https://tm-statweb.be/forms.php to test the data you send is correct.
